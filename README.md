# Smart Image Slider

A small JavaScript project that displays an image slider inside a single web page using HTML, CSS (Bootstrap), and vanilla JavaScript.

## Features

- Show image, caption, and current slide number.
- Navigate using **Prev** and **Next** buttons.
- Start/stop auto-play with **Play** and **Pause**.
- Add new slides by entering an image URL and caption.

## Technologies

- HTML5  
- CSS3 + Bootstrap 5  
- JavaScript

## How to Run

1. Clone or download this repository.  
2. Open `index.html` in any modern browser.  
3. Use the controls to browse slides or add your own images.

## Project Structure

- `index.html` – markup, styles, and script for the slider.  
- (Optional) `images/` – local images used in the slider.
